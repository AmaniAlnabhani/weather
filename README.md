# 🌤️ Weather App

A simple weather application that displays real-time weather 
information for any city, with a background that changes 
dynamically based on current weather conditions.

## Built With
- **Bootstrap** — Frontend 
- **PHP** — Backend logic
- **OpenWeatherMap API** — Weather data

## Features
- Search weather by city name
- Dynamic background based on weather condition
- Displays temperature, humidity, and weather description

## Setup
1. Clone this repository
2. Place it inside your `htdocs` folder (XAMPP)
3. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
4. Add your API key
5. Download background videos from (https://drive.google.com/drive/folders/14u048vMA1-Ozruw2YJImJ5ttoAPsCaff?usp=sharing) 
   and place them in `/videos`
6. Run via `localhost/weather-app`
